Stream preset

Choose how AutoStream decides what the best stream is. Normal preferences prioritize verified availability, seed health and a practical file size before resolution or premium formats.

Debrid

Use this preset with addons that already return cached debrid streams. AutoStream does not resolve debrid links itself yet.

Use host.docker.internal when Jackett runs on the same Docker host, or enter its LAN URL.

Use “all”, an indexer ID, or a Jackett filter such as status:healthy.
